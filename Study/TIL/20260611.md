#### ViewResolver
```
컨트롤러가 리턴한 뷰이름 앞에 접두사(prefix)와 접미사(suffix)를 붙여준다

<bean id="viewResolver" class="org.springframework.web.servlet.view.InternalResourceViewResolver">

<property name="prefix" value="/WEB-INF/board/"></property>

<property name="suffix" value=".jsp"></property>

</bean>
```

#### 쇼핑물 구매 버튼 클릭 시
```
//	@RequestMapping(value="/login.do", method = RequestMethod.GET)
	@GetMapping("/login.do") // 버전업
	public String loginView(UserVO vo) throws Exception {
		System.out.println("---> loginView() 실행");
		// Command 객체에 값을 설정하면 JSP 파일에서 사용할 수 있다
		vo.setId("admin");
		vo.setPassword("test123");
		return "login";
	}
	
//	@RequestMapping(value="/login.do", method = RequestMethod.POST)
	@PostMapping("./login.do") // 버전업
	public String login(UserVO vo, UserDAOJdbc dao) throws Exception {
		System.out.println("---> login() 실행");
		// 뷰이름 앞에 redirect:이나 forward:을 붙이면 ViewResolver가 동작하지 않는다
		if(dao.getUser(vo) != null) {
			return "forward:getBoardList.do";
		} else {
			return "login";
		}
	}
```
