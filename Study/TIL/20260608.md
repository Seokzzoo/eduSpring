#### pre-loading vs. lazy-loading
```
default -> pre-loading
pre-loading -> 메모리 낭비 but 속도가 빠름
lazy-loading -> 메모리 아낌 but 속도가 느림
```

#### IoC
```
역제어 <-> 순제어
뭘 제어할까? -> Object
Object -> 객체 생성, 의존 관계
순제어는 Java Code로 해야 함 -> 유지보수 극악
역제어는 Container가 함 -> xml만 수정하면 돼

★ JavaCode가 아닌 xml만 수정하자
```
